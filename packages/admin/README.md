# Allium Admin

Admin dashboard for the Allium framework.

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the admin dashboard.

## Features

- GraphQL integration via Apollo Client
- Model management interface
- Schema editor
- **Analytics Dashboard**: Performance monitoring with charts and custom date filtering
- Dynamic admin UI
